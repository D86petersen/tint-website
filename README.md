# Extreme Labors - Handyman & Remodeling Website

A professional website for Extreme Labors, a handyman and remodeling company featuring a modern contractor color scheme with yellow and orange.

## Features

- Modern, responsive design optimized for all devices
- Professional contractor color scheme (yellow #FDB714, orange #FF6B35)
- Services showcase section
- Company information and statistics
- Contact form for customer inquiries
- Smooth navigation and user-friendly interface

## How to View

Simply open `index.html` in a web browser, or host the files on any web server.

## Technologies Used

- HTML5
- CSS3 with CSS Grid and Flexbox
- Responsive design with mobile-first approach
